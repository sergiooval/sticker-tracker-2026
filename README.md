# Sticker Tracker 2026

Sitio web estático para llevar control de stickers pegados, faltantes y repetidos.

## Cómo usar
1. Sube todos estos archivos a GitHub Pages, Netlify o Vercel.
2. Abre `index.html` en el navegador.
3. En iPhone/Android puedes agregarlo a pantalla de inicio. El logo ya está incluido como icono.

## Notas
- El progreso se guarda localmente en el navegador del usuario.
- El botón “Exportar progreso” descarga un respaldo JSON.
- El botón “Modo cámara” abre la cámara o galería para usar la foto como referencia visual.
