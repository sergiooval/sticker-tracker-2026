# ⚽ Sticker Tracker 2026

Sticker Tracker 2026 is a modern web app for tracking a World Cup 2026 sticker album collection.

It helps collectors keep track of collected stickers, missing stickers, duplicates, and overall album progress from any device.

## Features

- Track collected stickers
- Mark stickers as missing, collected, or duplicate
- View album completion percentage
- See missing sticker count
- Manage duplicate stickers
- Organized by World Cup groups
- Includes special sections:
  - FIFA World Cup stickers
  - Coca-Cola stickers
- Mobile-friendly responsive design
- Works on iPhone, Android, tablets, and desktop
- Installable as a Progressive Web App (PWA)
- Export and import collection backup
- Camera/photo upload mode for album page reference

## Current Album Structure

- 48 national teams
- 20 stickers per team
- FIFA World Cup special stickers
- Coca-Cola special stickers

## How to Use

1. Open the website.
2. Search for a team or sticker code.
3. Tap a sticker once to mark it as collected.
4. Tap it again to mark it as duplicate.
5. Tap it again to reset it to missing.
